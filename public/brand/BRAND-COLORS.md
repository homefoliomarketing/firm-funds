# Firm Funds Brand Colors & Typography

## Primary Colors

| Name              | Hex       | Usage                                    |
|-------------------|-----------|------------------------------------------|
| Corporate Charcoal | #1E1E1E  | Logo, headings, primary text             |
| Dark Charcoal      | #2D2D2D  | Backgrounds, cards, secondary elements   |
| Mid Charcoal       | #3D3D3D  | Borders, subtle UI elements              |
| Warm Tan           | #B8A88A  | Accents, highlights, premium feel        |
| Silver             | #A8A8A8  | Secondary text, metallic accents         |
| Light Silver       | #C8C8C8  | Disabled states, subtle borders          |

## Background Colors

| Name              | Hex       | Usage                                    |
|-------------------|-----------|------------------------------------------|
| Off-White          | #F5F5F0  | Page backgrounds                         |
| Pure White         | #FFFFFF  | Cards, content areas                     |
| Charcoal Dark      | #1E1E1E  | Dark mode backgrounds, footer            |

## Status Colors

| Name    | Hex       | Usage             |
|---------|-----------|-------------------|
| Success | #2D7A3A  | Funded, approved  |
| Warning | #B8860B  | Under review      |
| Error   | #A82828  | Denied, errors    |
| Info    | #2D5A8A  | Submitted, info   |

## Typography

- **Headings**: Bold sans-serif (Inter Bold / Arial Black fallback)
- **Body**: Regular sans-serif (Inter / system-ui fallback)
- **Logo wordmark**: Extra bold, wide letter-spacing

## Logo Files

- `logo-mark-charcoal.svg` — Icon only, dark
- `logo-mark-white.svg` — Icon only, white (for dark backgrounds)
- `logo-full-charcoal.svg` — Icon + wordmark, stacked
- `logo-full-white.svg` — Icon + wordmark, white
- `logo-horizontal-charcoal.svg` — Icon + wordmark, side by side
- `logo-horizontal-white.svg` — Icon + wordmark, side by side, white
- `favicon.svg` — Favicon (charcoal bg, white mark)
- PNG versions at 64, 128, 256, 512px widths
